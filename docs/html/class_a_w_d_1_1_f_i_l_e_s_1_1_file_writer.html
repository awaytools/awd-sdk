Write a unsion to the file.

Writes a union according to the given TYPES::data_types and settings.

Parameters
[in]valThe union to write to file
[in]data_typeThe type of data to write
[in]data_lengthThe length of the data. For Numbers, this should be the length of the data as F64
[in]use_preciseIf true, Numbers are writen as F64 otherwise F32
[in]use_scaleIf true, the values writing to file will be scaled.
