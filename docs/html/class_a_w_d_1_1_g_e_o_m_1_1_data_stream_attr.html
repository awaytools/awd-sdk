Detailed Description

This is a single attribute of a SubGeom-Stream. is the class set can either be added to a Vertex or to a Triangle in order to save custom-attribute-streams.
This class is made as small as possible, because it must be present per Triangle or Vertex.
All properties of the Attr that are the same for all Vertex or Triangle are saved in a The DataStreamRecipe contains a DataStreamattrDescription object, that is used to further descripe the DataStreamAttr when writing the streams to file.
