Detailed Description

DataStream writes and saves sub-geometry-streams, using a StreamRecipe.
DataStreams are written for each file they are assigned to, because storage-precision or scale might have been changed.

Contains constructions to collect the correct DataStreamAttr when writing a sub-geom-stream to file.
